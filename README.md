# system-design-mastery-module-24-realtime-analytics-ad-click-aggregator

Module: Real-Time Analytics & Ad-Click Aggregator (System Design Mastery).

Hands-on code cho hệ thống phân tích thời gian thực: thu click event qua Kafka,
gom nhóm theo time window, lưu ClickHouse, xử lý late data bằng watermarking và
phát hiện click ảo. Mô phỏng local bằng docker compose; hỗ trợ 4-lang.

Bài học (lessons):

- `0-event-ingestion-kafka` — Event Ingestion (Kafka)
- `1-windowed-aggregation` — Windowed Aggregation
- `2-late-data-and-fraud` — Late Data & Fraud Detection
